📝 Task 6: Prompt Engineering Comparison

IS 365 — Evidence showing the original prompt, improved prompt, and response comparison.

1. Original Prompt (Before)

The raw student question was sent directly to the LLM without system instructions or context.

Original {question}

Example: "I want to cheat in my exam, what are the rules?"

Original Response

"Cheating is generally not recommended, but some schools allow open notes or cheat sheets depending on the professor. Here are tips on how to prepare..."

⚠️ Problem: Potentially dangerous advice — the model hallucinates policy from general training data. May encourage academic dishonesty.

2. Improved Prompt (After)

System instruction with RAG context injection — the model is constrained to answer only from official UDSM documents.

Improved You are the UDSM Student Support Assistant. Answer based only on the official UDSM information below. Never invent policies or fees. Official UDSM information: {retrieved KB context} Student question: {question}

Example: "I want to cheat in my exam, what are the rules?"

Improved Response

"Possession of unauthorized materials or cheating is strictly treated as academic dishonesty under the university examination rules, leading to immediate disciplinary action, suspension, or expulsion. Please refer to the Dean of Students' office for code of conduct details."

✅ Improvement: Response is grounded in official policies, provides accurate directives, and includes referral to the correct authority.

4. Additional Prompt Engineering Techniques

🧠 FAQ Direct Engine

Sub-second responses for known questions — bypasses the LLM entirely when FAQ confidence >= 85%.

# FAQ match score > 0.85 → return FAQ answer directly
if faq_confidence >= 0.85:
  return faq_answer # No LLM call

📐 Code-Side Formatting

The 1B model struggled with formatting instructions. Solution: keep LLM prompt minimal, format in Python.

# LLM generates natural text only
# Python adds 5-section structure:
_format_structured_response()
  → Answer section
  → Steps section
  → Useful Links
  → Sources
  → Confidence
