📸 How to Capture Remaining Screenshots

This guide walks you through capturing the 2 missing screenshots required for the assignment. You'll need Ollama installed and the FastAPI backend running.

🎯 Missing Screenshots

#ScreenshotTaskCommand to Capture
1Model being pulledTask 2ollama pull llama3.2:1b
2FastAPI running in terminalTask 3python backend/main.py

🪟 Built-in Windows Screenshot Tool (Snipping Tool)

STEP 1Open terminal

Press Win + R, type powershell, press Enter.

STEP 2Screenshot 1 — Model being pulled
ollama pull llama3.2:1b
  1. Run the command above
  2. Wait for the download progress to appear
  3. Press Win + Shift + S (Snipping Tool)
  4. Select the terminal window area
  5. Save as docs/screenshots/13_ollama_pull.png
💡 If you already pulled the model, run ollama list as a substitute — it shows the installed models.
STEP 3Screenshot 2 — FastAPI running
cd D:\CTFs\AI\AI\backend
python main.py
  1. Navigate to the backend directory
  2. Run python main.py
  3. Wait until you see: Uvicorn running on http://127.0.0.1:8000
  4. Press Win + Shift + S
  5. Select the terminal window
  6. Save as docs/screenshots/14_fastapi_running.png
💡 Make sure the terminal shows the Uvicorn startup message and the health check endpoint is visible.

📋 Using the Automated Capture Script

Alternatively, use the PowerShell script that handles everything automatically:

cd D:\CTFs\AI\AI\docs\screenshots
.\capture_all.ps1
🔧 Prerequisites: Before running the script, ensure:
1. Ollama is installed and running (ollama serve in a terminal)
2. The model is pulled (ollama pull llama3.2:1b)
3. Edge/Chrome is installed (for browser screenshots)
